# Changelog

All notable changes to Kyla Agent are documented here.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-05-07

### Added
- Skills Marketplace for discovering and managing agent skills
- Hacker Agent interface for security-oriented scanning workflows
- RepoScan dashboard for GitHub repository trust scoring
- Real-time UI panels for logs, findings, headers, network activity, and repository metadata
- Multi-panel interface with collapsible file tree, code viewer, and AI analysis
- Trust scoring with animated visualization (VERIFIED / TRUSTED / CAUTION / SUSPICIOUS / HIGH RISK)
- Share and export functionality with badge generation and CT thread export
- React 18 + TypeScript + Vite build system
- Tailwind CSS with custom design system
