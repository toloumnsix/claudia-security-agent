## Core Promise

Interrogate any GitHub repo. Expose truth. Prevent rugs. Verify builders.

## Features

### 3-Panel Intelligence Layout
- **File Tree**: Code similarity analysis with per-file percentage scores
- **Code Viewer**: Syntax highlighting with threat annotations
- **AI Analysis**: Trust score with animated visualization

### Trust Scoring System
- **VERIFIED** (85-100): High confidence, original code
- **TRUSTED** (70-84): Good standing, minimal concerns
- **CAUTION** (55-69): Some red flags detected
- **SUSPICIOUS** (40-54): Multiple issues found
- **HIGH RISK** (0-39): Significant threat indicators

### AI-Powered Analysis
- Code authorship detection (HUMAN/AI-GEN/COPIED/MIXED)
- Threat pattern recognition
- Commit DNA analysis
- Author intelligence profiling

### Ask Misaki
Inline chat interface for repository-specific questions with streaming responses.

### Share & Export
- Generate shareable links
- Create trust badges
- Export CT-ready thread summaries
- PDF report generation

## Tech Stack

- React 18 + TypeScript
- Vite build system
- Tailwind CSS
- Custom design system

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
