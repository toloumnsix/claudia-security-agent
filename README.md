## Features

### Skills Marketplace

Discover and manage agent skills for enhanced capabilities. Browse available skills, install with one click, and customize your agent's skill set.

## Tech Stack
<img width="1558" height="1215" alt="m5" src="https://github.com/user-attachments/assets/cff14d29-005e-4303-8c0e-800ec37ed2f0" />

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI + shadcn/ui patterns
- **Charts**: Recharts
- **API Integration**: GitHub REST/GraphQL API

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
