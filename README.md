## Features

### Skills Marketplace
Discover and manage agent skills for enhanced capabilities. Browse available skills, install with one click, and customize your agent's skill set.

### Hacker Agent
Security-oriented scanning interface with real-time threat detection. Identify vulnerabilities, analyze code patterns, and generate comprehensive security reports.

### RepoScan
GitHub repository trust scoring dashboard. Evaluate repository integrity with scoring metrics based on code authorship, commit patterns, and community signals.

### Real-time Dashboard
Multi-panel interface displaying:
- Logs and findings in real-time
- Header and metadata information
- Network activity monitoring
- Repository intelligence metrics

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI + shadcn/ui patterns
- **Charts**: Recharts
- **API Integration**: GitHub REST/GraphQL API

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
